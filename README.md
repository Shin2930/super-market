โปรเจกต์นี้ขับเคลื่อนด้วยสถาปัตยกรรมแบบ MVC (Model-View-Controller) เพื่อแยกสัดส่วนการทำงานให้เป็นระเบียบ โดยมี `productModel.js` ทำหน้าที่เชื่อมต่อและจัดการข้อมูลในระบบฐานข้อมูล MySQL มีไฟล์ `index.ejs` และ `add-product.ejs` เป็นส่วนของ View ที่คอยแสดงหน้าต่างอินเตอร์เฟซให้ผู้ดูแลระบบใช้งาน และมี `productController.js` ทำหน้าที่เป็นตัวกลางคอยรับคำสั่งจากผู้ใช้เพื่อไปประสานงานร่วมกับฝั่ง Model ต่อไป

สำหรับการเริ่มต้นใช้งาน ให้เปิดโปรแกรมจัดการฐานข้อมูล เช่น XAMPP หรือ phpMyAdmin เพื่อสร้างฐานข้อมูลใหม่ขึ้นมาโดยตั้งชื่อว่า `supermaket` จากนั้นให้ทำการนำเข้า (Import) ไฟล์ `supermaket.sql` เพื่อสร้างตาราง `products` สำหรับจัดเก็บข้อมูลสำคัญของสินค้า ได้แก่ ชื่อสินค้า หมวดหมู่ ราคา จำนวนสต็อก และชื่อไฟล์รูปภาพ เมื่อจัดการฝั่งฐานข้อมูลเรียบร้อยแล้ว ให้เข้าไปตรวจสอบและตั้งค่าการเชื่อมต่อในไฟล์ `productModel.js` โดยระบุ Host เป็น `localhost`, User เป็น `root`, เลือกฐานข้อมูล `supermaket` และใส่รหัสผ่าน MySQL ของตนเอง (หากไม่มีให้ปล่อยว่างไว้)

ขั้นตอนสุดท้ายในการเปิดทำงาน ให้เปิด Terminal ขึ้นมาในโฟลเดอร์โปรเจกต์แล้วรันคำสั่ง `npm install` เพื่อทำการติดตั้ง Library และ Dependencies ทั้งหมดที่จำเป็น หลังจากนั้นให้สั่งเริ่มระบบด้วยคำสั่ง `node app.js` ซึ่งหากเซิร์ฟเวอร์พร้อมใช้งานจะมีการแจ้งเตือนข้อความยืนยันการเชื่อมต่อบนพอร์ต 3000 จากนั้นผู้ใช้ก็จะสามารถเข้าใช้งานระบบผ่านเบราว์เซอร์ได้ทันทีที่ลิงก์
